# React Notes App

A simple Notes App built using React and Tailwind CSS. This project allows users to create and manage notes through a clean and responsive interface.

## Features

* Add notes with a title and description
* Form validation for empty inputs
* Dynamic rendering of notes using React state
* Real-time notes counter
* Responsive UI built with Tailwind CSS

## Technologies Used

* React
* JavaScript (ES6+)
* Tailwind CSS
* Vite

## Concepts Practiced

* React Components
* JSX
* useState Hook
* Event Handling
* Two-Way Binding
* Controlled Components
* Array State Management
* Dynamic Rendering with map()

## Future Improvements

* Delete Notes
* Search Functionality
* Edit Notes
* Local Storage Support
* Dark Mode

## Getting Started

1. Clone the repository
2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```
